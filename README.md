# DEF JAM FIGHT FOR NY — WIKI

A fan-made wiki tribute to the legendary 2004 hip-hop fighting game.

## Structure

```
defjam-wiki/
├── index.html          ← Main page
├── css/
│   ├── style.css       ← Core styles & layout
│   └── animations.css  ← Keyframes & motion effects
├── js/
│   └── main.js         ← Scroll animations, interactions
└── README.md
```

## Features

- Dark street / hip-hop brutalist aesthetic
- Glitch text effects on the main title
- Scroll-reveal animations throughout
- Cursor trail effect
- Animated style power bars
- Seamless marquee dividers
- Parallax header
- Animated score counter
- Sections: Overview, Gameplay, Fighters, Venues, Soundtrack, Legacy

## Usage

Just open `index.html` in any modern browser — no build tools or dependencies required.
